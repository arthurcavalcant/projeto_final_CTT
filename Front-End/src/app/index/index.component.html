Observações:

Esta aplicação simula um Sistema Imobiliário, as funcionalidades aqui dispostas, até o momento, são: Cadastro de clientes, proprietários, imóveis, vendedores e as vendas efetuadas. Algumas dessas entidades permitem atualizar informações, após clicar no botão "Alterar", caso haja o botão na entidade, você verá os campos disponíveis para alteração. No momento, a exclusão de uma entidade que tenha relação com outra acarretará na exclusão em cascata, por exemplo: um imóvel que possui o proprietário de ID 2, caso esse proprietário seja excluído o imóvel também será. O sistema atual permite que um mesmo CPF e RG seja um cliente, vendedor e proprietário ao mesmo tempo. Dentre as features planejadas para implementação, ainda há questões relacionadas à regra de negócio, como o impedimento de uma venda onde o cliente e o vendedor são a mesma pessoa, por exemplo.
